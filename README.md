# TurtleBot3 ML

A collection of Machine Learning projects built on top of the 
[TurtleBot3](https://github.com/ROBOTIS-GIT/turtlebot3) ROS 2 platform.

## Projects

| # | Package | Description | Status |
|---|---------|-------------|--------|
| 1 | `tb3_perception` | Object detection & person following | 🔄 In Progress |
| 2 | `tb3_control_ml` | Line following & gesture control | ⏳ Planned |
| 3 | `tb3_rl_navigation` | DQN & RL-based navigation | ⏳ Planned |
| 4 | `tb3_semantic_mapping` | Semantic mapping | ⏳ Planned |
| 5 | `tb3_imitation` | Imitation learning | ⏳ Planned |
| 6 | `tb3_sim_to_real` | Sim-to-real transfer | ⏳ Planned |
| 7 | `tb3_autorace_ml` | Autonomous race | ⏳ Planned |
